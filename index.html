Weather Wallah

Your Simple Weather Companion

Weather Wallah ek simple aur powerful weather app hai jo aapko sab kuch ek jagah deta hai — live weather, forecasts, air quality, map, aur smart suggestions. Banaya hai Uday Deshmukh ne, dil se.

Project Highlights

Live Weather

Real-time temperature, humidity, wind, UV index — sab live data milta hai OpenWeatherMap API se.

Satellite Map

Interactive Leaflet.js map with ArcGIS satellite tiles. Location pin ke saath dikhta hai.

Air Quality

WAQI API se live AQI data. Color-coded labels — green se lekar red tak, hawa ki quality check karo.

7-Day Forecast

Weekly aur hourly forecast with weather icons. Kab barish hogi, kab dhoop aayegi — sab pata chalega.

What to Wear

Temperature ke hisaab se kapdon ki suggestion — garmi mein lightweight clothes, thandi mein warm layers.

What to Eat

Mausam ke according food suggestions — garmi mein smoothie bowl, thandi mein hot soup aur chai.

Voice Search

Mic dabaao aur city bolo — hands-free search kaam karta hai. Baar-baar type karne ki zaroorat nahi.

Mobile Ready

Bottom navigation, responsive design, fast loading — kisi bhi phone pe smooth chalta hai.
